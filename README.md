# 🧠 MindPulse — Mental Health Tracker Wearable

MindPulse is a modern, design-focused landing page for a **mental health wearable + companion app**. It showcases a product that tracks stress, sleep, and emotional wellbeing using biometric data, and presents insights in a calm, user-friendly interface.

---

## 🌐 Overview

This project is a **responsive product landing page** built using pure HTML, CSS, and a little JavaScript. It is designed to:

* Present a wearable mental health product
* Highlight features and benefits clearly
* Provide a visually engaging UI
* Encourage user conversion (pre-orders / signups)

---

## ✨ Features

### 🎯 UI/UX Highlights

* Clean, modern design system (mint, lavender, coral palette)
* Serif + sans-serif typography pairing
* Soft shadows, rounded cards, and subtle animations
* Fully responsive layout (desktop → mobile)

### 🧩 Sections Included

* **Navigation bar** (sticky)
* **Hero section** with product positioning
* **Interactive device mockup**
* **Features grid**
* **How it works** (step-by-step)
* **Statistics / social proof**
* **Testimonials**
* **Call-to-action (CTA)**
* **Footer**

### ⚙️ Interactivity

* Mood selection buttons (UI state change)
* Smooth scrolling navigation
* Hover animations on cards and buttons

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**

  * Flexbox & Grid
  * Custom properties (CSS variables)
  * Responsive media queries
* **Vanilla JavaScript**

  * DOM manipulation
  * Event listeners

No frameworks or libraries are used.

---

## 📱 Responsive Design

The layout adapts for:

* Desktop (full grid layout)
* Tablet (reduced columns)
* Mobile:

  * Hero becomes single-column
  * Device mockup hidden
  * Navigation simplified

## 🎨 Design Philosophy

MindPulse is designed around:

* **Calmness over clutter**
* **Clarity over complexity**
* **Emotional connection through UI**

The visual language reflects mental wellness:

* Soft colors → reduce cognitive load
* Rounded edges → feel approachable
* Minimal data → avoid overwhelm
